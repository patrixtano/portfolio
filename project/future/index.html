The Problem

We carry many possible futures, but no way to find the one that belongs to you.

The Solution

A fortune prediction installation. Users submit a word via a captive WiFi portal on a Raspberry Pi, which finds a semantically matching Czech fortune and prints it on a thermal printer. This page visualizes every prediction ever discovered.
